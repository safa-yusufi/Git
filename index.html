Git Tutorial

steps of working with git:

  1. first of all check the git version to see the version of your git in cmd or in your terminal if you do not have git in your pc install it.
  2. now let git know who you are :Change the user name and e-mail address to your own.
  3. Now, let's create a new folder for your project
  4. Once you have navigated to the correct folder, you can initialize Git on that folder(Git creates a hidden folder to keep track of changes.)
  5. You just created your first local Git repo. But it is empty. So let's add some files, or create a new file using your favourite text editor. Then save or move it to the folder you just created.
  6. Then,Check the status of the Git ()git status
  7. Next,The file should be Staged.(git add .)
  8. again chek the status, Now the file has been added to the Staging Environment
  9. Since we have finished our work, we are ready move from stage to commit for our repo
  10. When we commit, we should always include a message.(The commit command performs a commit, and the -m "message" adds a message.)
  11. To view the history of commits for a repository, you can use the log command.
  12. now you shoould create an account on Github,then sign in and Create a Repository on GitHub
  13. After creating a new repository,Since we have already set up a local Git repo, we are going to push that to GitHub
  14. git remote add origin URL specifies that you are adding a remote repository, with the specified URL, as an origin to your local Git repo.
  15. Now we are going to push our master branch to the origin url, and set it as the default remote branch.
